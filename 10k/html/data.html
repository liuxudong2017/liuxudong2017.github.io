    课程大纲
    时间安排
    平均每周5小时基于视频会议进行在线培训
    第1周
    数据分析入门
    • 数据分析师招聘情况与职业发展
    • BAT TMD数据分析师招聘JD分析
    • 主流互联网公司数据流转的过程
    • 数据分析师的职责以及与其它职能的关系
    • 若干数据分析师自身成长经历与当前的工作分享
    • 什么样的人能成为优秀的数据分析师？
    第2-3周
    SQL语言介绍与实践
    • SQL基本语法
    • 函数与操作符、过滤与排序
    • Union Join 与子查询
    • 数据分组（Group By）
    • Hive 与Mysql的SQL语法差异
    第4周
    Excel技能培训
    • Excel基本操作与常用函数
    • VLOOKUP和数据透视表
    第5周
    描述性统计分析
    • 什么是描述统计学？
    • 集中趋势(Center)：平均值、中位数、众数
    • 离散程度(Dispersion)：标准差、方差、百分位数、次位指标
    • 分布形态 (Shape) : 峰值、偏度
    • 数据可视化
    • 常用数据分析方法：队列分析（Cohort）、漏斗转化分析、对比分析

     

    第6周
    推论统计与假设检验
    • 置信区间、区间估计、显著性差异
    • 检验方法：z检验、t检验、f检验、卡方检验
    • A/B 测试的统计学原理
    第7周
    Python 极简入门
    • 变量类型与运算符
    • 条件语句与循环语句
    • List/Tuple/Dict/Set
    • 字符串操作、文件操作
    • Python后续学习指引
    第8周
    Python数据分析
    • Python数据的描述性统计
    • 方差分析
    • 假设检验
    • 数据可视化
    第9 - 11周
    数据分析的应用与实践
    • Growth Hacking的数据分析框架
    • 用户行为的数据分析
    • 用户的价值体系（CAC/LTV/RFM）
    • 广告与在线营销数据分析（CPM/CPC/CPA/转化率）
    • A/B测试
    第12周
    求职辅导
    • 简历撰写指导
    • 1v1 模拟面试
    • 求职过程跟踪与机会推荐
